# Security Policy

## Reporting a Vulnerability

**Please do not file public GitHub issues for security vulnerabilities, and do
not post them in public Discord channels.**

Use one of these private reporting channels:

1. **GitHub Private Vulnerability Reporting** (preferred) — open a private
   advisory at
   [github.com/getcodesetu/codesetu/security/advisories/new](https://github.com/getcodesetu/codesetu/security/advisories/new).
2. **Direct message a maintainer** in our
   [Discord server](https://discord.gg/sjVKU8cpC6). Send a DM — do not post
   vulnerability details in any public channel.

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- (Optional) Suggested fix

You will receive an acknowledgement within 48 hours and a resolution timeline
within 7 days.

## Supported Versions

| Version        | Supported              |
| -------------- | ---------------------- |
| main branch    | ✅                     |
| older releases | ❌ (upgrade to latest) |
